# Southeast Media Website 

Production foundation for a premium corporate website built with Next.js App Router, React, TypeScript, Tailwind CSS, GSAP, Framer Motion, Lenis, shadcn/ui conventions, Lucide React, React Hook Form, Zod, ESLint, Prettier, Husky, and lint-staged.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run lint` runs ESLint.
- `npm run typecheck` runs TypeScript checks.
- `npm run format:check` checks formatting.
- `npm run lint-staged` runs staged-file checks.

## Foundation Scope

This sprint includes architecture, configuration, reusable primitives, design tokens, providers, metadata, placeholders for robots and sitemap, motion utilities, Lenis smooth scrolling, and global accessibility defaults. Public content pages, hero, services, and portfolio sections are intentionally deferred.
